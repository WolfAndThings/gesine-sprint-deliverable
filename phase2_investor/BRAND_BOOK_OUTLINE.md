## Production notes

- Print resolution if printed (300dpi) and screen-optimized version (72dpi web)
- File format: InDesign master, PDF export, plus Google Slides version for live edits
- Fonts: license check before publishing (paid licenses for serif + sans)
- Photography: every image either licensed, owned, or AI-generated with brand-permitted prompts. NEVER stock without license. NEVER AI-generated without explicit brand approval.

## What this document is NOT

- It is NOT the executed brand book PDF. That is a designer pass after palette is locked.
- It is NOT the manifesto or origin story. Lee writes those.
- It is NOT the photo shoot. That is its own production.

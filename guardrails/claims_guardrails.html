◆ The legal & brand line we never cross
What we can say.
What we must prove.
What kills the brand.
Premium home & wellness brands collapse not from bad design but from one stray claim — "kills 99.9%", "non-toxic", "cures stress". The FTC and FDA Green Guides are the iceberg most challengers hit. This document is the rule book for every line of GESINE copy: three claim tiers, eight substantiation buckets, five before/after rewrites, and the regulatory flags Marketing must check before shipping any creative.
◆ Three-tier claim system

Safe. Conditional. Off-limits.

Every claim falls into one of three buckets. Use the SAFE list anywhere. Use CONDITIONAL only when a substantiation file is on hand. Never use the OFF-LIMITS list, even by paraphrase.

◆ Substantiation files

Eight files, one folder.

Before launch, GESINE legal/ops keeps a single "Claims Substantiation" folder — eight buckets below. Every claim copy uses must point to a doc in this folder. If the doc isn't there, the claim isn't ready.

FILE 01
Full ingredient disclosure (INCI)
Complete INCI list per SKU, with %-by-weight where shareable. Required for D2C site footer + retail label compliance + EWG submission.
FILE 02
Fragrance disclosure
Per-scent breakdown of IFRA-compliant fragrance components. Allergens called out per EU 1223/2009 even if shipping US-only. Backs every "scent-led" and "fragrance from X" claim.
FILE 03
EWG verification documents
EWG application + supporting raw-material safety dossiers. Without "EWG VERIFIED" certification, only "EWG-conscious" or "screened against EWG" is permissible.
FILE 04
Safety Data Sheets (SDS)
OSHA-compliant 16-section SDS per SKU. Required for every B2B and hospitality customer. Hazard pictograms, transport class, first-aid responses.
FILE 05
Biodegradability substantiation
OECD 301B / 310 test results from a third-party lab. Required before any "biodegradable" claim ships in copy or on label. Specifies time-to-degrade and conditions.
FILE 06
Packaging specifications
Bottle · refill cartridge · outer carton material specs, % recycled content, recyclability path. Backs plastic-free, refillable, recyclable claims and FTC Green Guide compliance.
FILE 07
Surface compatibility testing
Documented testing on marble, wood, lacquer, stainless, glass, ceramic. Without this file, do not claim "safe for [surface]". Required for retail buyer due-diligence.
FILE 08
Approved claims index
A single master index of every approved claim, mapped to its substantiation file and last-reviewed date. Marketing must check this before any ad spend, organic post, or PR pitch ships.

◆ FTC + FDA flag list

Where cosmetics meet cleaning. Where brands collapse.

GESINE sits on a regulatory knife-edge: a cleaning product that smells, looks, and is sold like fragrance. The crossover invites scrutiny from both the FTC (advertising) and the FDA (cosmetic / drug crossover). Each flag below is a known landmine.

FTC
Green Guides · vague environmental claims. "Eco-friendly" / "sustainable" without a specific qualifier triggers FTC Green Guide enforcement. Always pair with the specific claim ("plastic-free packaging", "concentrated formulation").
FTC
Comparative claims. "Better than [Lysol]" requires a documented test that supports the comparator on the same metric. Without it, the comparator must be removed or made qualitative ("designed differently").
FTC
Influencer / UGC compliance. Every paid creator must use #ad or "Paid partnership" tag. Brand is liable for creator's claims — brief them with the safe-tier list, not vague "say what you love."
FDA
Antibacterial / disinfectant claims. Triggers FDA / EPA jurisdiction (depending on intended use) — the product would have to be registered as a pesticide. Avoid every word in this family: kills, sanitize, disinfect, antibacterial, antiviral.
FDA
Drug claims via fragrance. Aromatherapy claims that imply mood or health effects ("relieves stress", "lowers anxiety", "helps you sleep") move the product into the drug category. Use sensory language only ("calming scent", "designed for end-of-day").
FDA
"Hypoallergenic" / "for sensitive skin". No FDA standard exists, but consumer-facing dermatologic claims still require independent skin-irritation testing on file. Without it, do not claim.
FTC
"Made in USA". Strict FTC standard: "all or virtually all" parts & processing must be domestic. If the bottle is imported, the claim must be qualified ("designed in USA, assembled in USA from imported components").
FTC
"Carbon neutral" / offsets. Must specify scope (Scope 1/2/3) and method (renewable energy, certified offsets). Vague carbon-neutral claims are now actively litigated as greenwashing.
◆ Pre-publish checklist
Before any ad, post, label, or press line ships: (1) Is every claim in tier SAFE or CONDITIONAL? (2) If conditional, is the substantiation doc on file and current? (3) Does the copy avoid every off-limits phrase verbatim and in paraphrase?

Three yeses — ship it. Anything else — rewrite.

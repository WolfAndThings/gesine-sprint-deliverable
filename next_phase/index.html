To kick all of this off · what we need from you

Three things unlock the rest.

Each item below is a checkbox. The day all three land, the polish-pass clock starts on the 3 finals and the Weavy handover, and the Phase 3 scoping work can move from this page to a signed line-item invoice. Nothing here is creative judgment, just material we need to do the work.

Need 01 · the geometry anchors

Detailed product photos.

Spray bottle: 4-6 angles, plain background, even light, no styling. Box: open + closed + dieline / flat artwork. Vials: single front + 3/4 + grouped. iPhone is fine, just consistent. The detailed shots become the lock reference the AI composite stops mutating against.

Need 02 · the visual library

Your reference boards.

Pinterest boards, screenshots, Slack threads, Google Drive folders, anything you've been collecting that you gravitated toward. Invite to the Pinterest board ideal. The faster I see what you keep saving, the faster the next round of content + the website pulls from the same visual gravity instead of guessing.

Need 03 · for the Shopify lane

Shopify admin access.

Staff account on the existing Prestige-template store so the Phase 3 build doesn't have to start from scratch. Read-only is enough to scope. Full edit access only when you say go on the rebuild. Plus your ConvertKit / Klaviyo account credentials if either is already running.

Need 04 · for the social lane

Voice + posting samples.

Any past social you've drafted (even unposted), any brand voice docs you've written, any captions you wish were yours. Faster to mirror your voice than invent one cold. If you have brand-tone words (calm, ritual, modern, etc.) saved anywhere, send those too.
